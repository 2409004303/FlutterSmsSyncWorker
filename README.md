### FlutterSmsSyncWorker

**(Android/IOS)实现短信内容实时拦截并上传服务器并且能查看云端短信内容的Flutter APP.**

功能：

- 用户登录注册(手机号)
- 短信内容实时拦截并上传服务器(只支持安卓)
- 查看属于自己(手机号)的云端短信内容

#### 服务端程序

- 在`server`文件夹中，使用[deployd](https://github.com/deployd/deployd)进行部署
